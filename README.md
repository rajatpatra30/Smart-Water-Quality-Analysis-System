# Smart Water Quality Monitoring & Analysis System

## 📌 Project Overview
This project analyzes water quality using chemical parameters such as aluminium, ammonia, arsenic, nitrates, and fluoride.

## 🚀 Features
- Data Cleaning & Preprocessing
- Water Quality Index (WQI) Calculation
- Classification (Safe / Moderate / Unsafe)
- Data Visualization using Matplotlib
- Identification of Highly Polluted Samples

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib

## 📊 Output
- Graphs showing pollution distribution
- Water classification system
- Processed dataset

## 🎯 Purpose
To analyze real-world water quality data and provide insights for environmental monitoring and decision-making.

- water_quality.py
- water_quality.csv
